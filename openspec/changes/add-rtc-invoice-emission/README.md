# add-rtc-invoice-emission

Emissão de notas fiscais no layout da Reforma Tributária do Consumo (RTC) — grupo IBS + CBS (e, para produto, também o Imposto Seletivo / IS) — via recursos dedicados `client.service_invoices_rtc` (NFS-e) e `client.product_invoices_rtc` (NF-e/NFC-e — modelos 55 e 65) no SDK Ruby v1.
