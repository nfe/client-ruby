# add-lookup-resources

Os 8 recursos de lookup/consulta/dados auxiliares do SDK Ruby v1 — addresses (CEP), legal_entity_lookup (CNPJ), natural_person_lookup (CPF), product_invoice_query + consumer_invoice_query (consulta por chave de acesso), tax_calculation, tax_codes e state_taxes — cada um roteado ao seu host correto pelo host map de `Nfe::Configuration`.
